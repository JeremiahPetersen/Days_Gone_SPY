Git Hub Contributing


1. Fork Repository
2. copy https of repo
3. open terminal (cd desktop)
4. git clone (https copied)
5. cd (folder name)
6. git init
7. git branch (name branch)
8. git checkout (branch name)
9. go to vscode folder
10. make changes
11. git add .
12. git commit -m "YOUR MESSAGE HERE"
13. git checkout main
14. git merge (branch name)
15. git remote add (branch name) https path(that has ur username) ,not the one u have forked from someone
16. git push -u (branch name) main
17. create new pull request
18. you should see changes made in your forked repo
